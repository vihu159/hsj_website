# How to run the HSJ website

**You must be inside the project folder before running npm.**

In Terminal:

1. Go to the project folder:
   ```bash
   cd ~/Desktop/HSJ_NEW_WEBSITE
   ```

2. Install dependencies (first time only):
   ```bash
   npm install
   ```

3. Start the dev server:
   ```bash
   npm run dev
   ```

4. Open in your browser: **http://localhost:3000**

If you see "Open Admin Panel" and "View Website", click **Open Admin Panel** to use the backend, or **View Website** to see the public site.
